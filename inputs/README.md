# chatbot_dio_exp2

Trata-se de um chat interativo que responde com base no conteúdo de arquivos PDF. Para isso, utilizaremos conceitos de IA generativa, embeddings e buscas vetorizadas para estruturar um sistema capaz de entender, processar e responder perguntas a partir de documentos específicos.

Essa abordagem me permitirá desenvolver habilidades para criar um modelo personalizado de assistência virtual focado em um conjunto de informações proprietárias, sem depender unicamente do conhecimento geral de modelos pré-treinados.

Cenário do modelo - auxiliar estudante de engenharia de software a estruturar conteúdo com base nas fontes de conhecimento inseridas no modelo (artigos científicos e matérias publicadas) sobre uso de AI em testes de software e programação.
